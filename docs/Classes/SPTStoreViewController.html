Overview

A Store View Controller to allow a user to download the Spotify iOS app.

To present the store controller on top of your view controller:

@code SPTStoreViewController *storeVC = [[SPTStoreViewController alloc] initWithCampaignToken:@“your_campaign_token” storeDelegate:self]; [self presentViewController:storeVC animated:YES completion:nil]; @endcode
