SPErrorWrongAPIVersion

The library could not be initialized because of an incompatible API version.

When calling SpInit(), you are required to set the field SpConfig::api_version to SP_API_VERSION. This error indicates that the library that the application is linked against was built for a different SP_API_VERSION. There might be an issue with the include or library paths in the build environment, or the wrong SDK shared object is loaded at runtime.
